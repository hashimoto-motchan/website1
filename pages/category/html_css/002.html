preタグとcodeタグ使用時に左にスペースが空いてしまう時の対処法

結論

テキストエディタに記載されているコードのインデント（字下げ）を削除する（左に詰める）

解説

コードを画面そのままに反映させるにはpreタグとcodeタグが便利です。

preタグとcodeタグはテキストエディタに記載されているコードのインテンドを反映してしまいます。

普通に行った場合（インテンドあり）

実際のコード（VSコードを使用）

                            
                                <head>
                                    <link rel="stylesheet" href="〇〇.css">
                                </head>

インテンドありのコードで表示されるコード画面（preとcodeを使用）

↑インテンドがそのまま反映されてしまい、左にスペースができてしまう。スクロールしないと見れなくてユーザビリティが低い。

インテンドを削除した場合

実際のコード

    
        <head>
            <link rel="stylesheet" href="〇〇.css">
        </head>

インテンドを削除したコードで表示されるコード画面（preとcodeを使用）

↑インテンドを削除することによってその分左に移動しました。


※上記のコード画面はprismを使用。preとcodeタグを調べると出来ました。
